# Hermes A-Share Golden Climbing — 黄金坑策略

> 范式转换: 从"猜谷底"到"统计深度-收益分布"  
> 预测问题 → 保险精算

## 策略全貌

| 环节 | 规则 |
|------|------|
| **选票** | 趋势得分 4-5/5 + C1(60日涨幅>全市场P60≈20%) + C4(涨幅>跌幅×2) |
| **闸门** | CSI300 > MA60 |
| **过滤** | 跌速 > 0.5%/天 + 量比 < 3x |
| **进场** | 跌到 -10%, T+1 开盘买入 |
| **出场** | 不止损不止盈, 持有 50-60天 到期平仓 |

**全部参数数据驱动，零拍脑袋。**
